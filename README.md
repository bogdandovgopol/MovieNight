# Watch Room
MovieNight is a Swift iOS app showing currently playing and upcoming movies using data from themoviedb.org's REST api.

You can try the app at the [App Store](https://apps.apple.com/us/app/id1520364631).


## Dependencies
Firebase
Kingfisher
